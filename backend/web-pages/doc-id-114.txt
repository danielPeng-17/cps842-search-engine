PHOON - SHOW THE CURRENT PHASE OF THE MOON

Shows the moon in its current phase, including the partial lighting of the dark side by reflected earthlight. A Java version of xphoon
[http://www.acme.com/software/xphoon/], which is itself an X version of phoon [http://www.acme.com/software/phoon/].
